# Chess Opening Study Program

My attempt at a chess opening program.

Program to work on chess openings. Single lines can be written as:
1. e4 e5 2. Nc3 Nf6 3. f4 exf4 4. e5 Ng8 5. Nf3 d6 6. d4 dxe5 7. Bb5+ c6 8. Bc4 *
